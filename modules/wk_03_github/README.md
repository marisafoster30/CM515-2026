# Week 3: GitHub and Reproducibility
Instructors: Dmitri Svetlov and Kira Vasquez-Kapit

## Learning Objectives and Significance

There is a reproducibility crisis in science, particularly when it comes to data processing/data analysis/computational work. This module will highlight the importance of proper documentation and methods when coding, and will provide students with practice organizing and documenting projects via the version control system Git and the corresponding platform GitHub.

## Session 1, Monday Feb. 2
Students will become familiar with: 
* The importance of reproducibility
* What version control means and how it pertains to reproducibility
* The fundamentals of Git basic commands
* Practice basic operations through GitHub and GitHub Desktop such as forking, commits, pushing and pulling, etc.
* Navigating local and remote repositories

### Assignments
Three submissions related to in-class activities to gain practice with Git concepts and operations.
